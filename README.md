# George — Every Word Has Weight

Personal website and short-form blog for publishing brief, heart-gripping reflections, lessons, and articles.

## Structure

- `html/home.html` - main site page
- `css/home.css` - site styles
- `js/home.js` - site interactions
- `assets/images/` - images and visual assets
- `.instructions.md` - project-specific content and development instructions

## Content Direction

- Short reads, typically around 1 minute
- Simple, clear, emotionally weighty writing
- Focused on thoughts, lessons, reflections, and material worth reading
- Layout preserved as designed while content evolves

## Local Use

Open `index.html` in a browser, or open `html/home.html` directly.
